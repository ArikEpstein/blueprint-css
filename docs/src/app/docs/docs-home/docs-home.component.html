Build great responsive mobile first UIs that work everywhere with Blueprint CSS. Free and Open Source based on Flexbox.

Getting Started

To get started install via npm.

npm install blueprint-css --save

Alternatively copy and paste the stylesheet <link> into your <head> to load the CSS.

<link href="https://unpkg.com/blueprint-css@{{blueprintVersion}}/dist/blueprint.min.css" rel="stylesheet" />
