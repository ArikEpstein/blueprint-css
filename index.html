Core Flex Grid

Core Flex Grid is a modern responsive mobile first grid built on top of Flexbox.
Core Flex Grid is lightweight weighing at only 1.5kb minified and gziped. Open Source on GitHub. IE10+ support.

Utility Classes

Containing Classes (max-width)

Using containing classes you can contain elements width proportionally to the max width of the .container class. This allows containers to be created that are flexible but do not scale with the view port. Example if we add a .contain--6 the element max width will be half of the .container class width. (1000/2 = 500px). This can be useful for setting a max width on objects such as forms.

.contain--8 (max-width: 733.33333px)

.contain--6 (max-width: 550px)

.contain--4 (max-width: 366.66667px)

Layout Helpers

Layout helpers are for elements or small modules that only need a small layout change such as display: inline; to display: block;. Layout helper classes should not be used on modules or elements that have Sass modules or classes already applied. 


/* --- Layout helpers --- */
.display-none {
    display: none;
}
.display-none-important {
    display: none !important;
}
.display-block {
    display: block;
}
.display-inline-block {
    display: inline-block;
}
.visible-hidden {
    visibility: hidden;
}
.block-center {
    margin-left: auto;
    margin-right: auto;
    display: block;
    float: none;
}
.float-left {
    float: left;
}
.float-right {
    float: right;
}
.full-width {
    width: 100%;
}
.clear-both {
    clear: both;
}
      
/* Margin and Padding Helpers */
.margin-top {
    margin-top: 24px;
}
     
.margin-bottom {
    margin-bottom: 24px;
}
     
.margin-top-small {
    margin-top: 12px;
}
     
.margin-bottom-small {
    margin-bottom: 12px;
}
     
.padding-bottom-none {
    padding-bottom: 0;
}
