Core Flex Grid

Core Flex Grid is a modern responsive mobile first CSS grid & layout library built on top of Flexbox.
Core Flex Grid is lightweight weighing at only 1.5kb minified and gziped. Open Source on GitHub. IE10+ support.

Gutter Spacing

With special modifiers you can remove gutters in flex-row layouts. .flex-row--gutter-none .flex-row--gutter-bottom-none .flex-row--gutter-between-none Example with .flex-row--gutter-bottom-none we can remove the bottom gutter from our flex-row. This is usefull for rows that have custom padding or spacing and need to override the grids default.

Max Classes (max-width)

Using max classes you can set the max width on elements proportionally to the max width of the .container class. This allows containers and elements to be created that are flexible but do not scale with the view port. Example if we add a .col-6--max the element max width will be half of the .container class width. (1000/2 = 500px). This can be useful for setting a max width on objects such as forms or flexible images. Max classes can be used anywhere and are not restricted to only in .flex-row.

.col-8--max (max-width: 733.33333px)

.col-6--max (max-width: 550px)

.col-4--max (max-width: 366.66667px)

Utility Classes

Utility classes are for elements or small modules that need a minor layout adjustments. 


.float-center {
    margin-left: auto;
    margin-right: auto;
    display: block;
    float: none;
}

.float-left {
    float: left;
}

.float-right {
    float: right;
}

.clear-fix:after {
    content: "";
    display: table;
    clear: both;
}

.full-width {
    width: 100%;
}

.text-left {
    text-align: left;
}

.text-right {
    text-align: right;
}

.text-center {
    text-align: center;
}

.text-justify {
    text-align: justify;
}
