# Core Flex Grid
Responsive CSS Flexbox Grid
 
<img src="https://raw.githubusercontent.com/splintercode/core-flex-grid/master/assets/images/icon.png" alt="Core Flex Grid" style="width: 100%; max-width: 120px;" />

Core Flex Grid is a modern responsive mobile first grid built on top of Flexbox.
Core Flex Grid is lightweight weighing at only 2kb minified and gziped.
IE10+ [support](https://splintercode.github.io/core-flex-grid/#bowser-support).
